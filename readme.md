# 🌿 Sylheti Programming Language (SPL) v1.0.0

**Sylheti Programming Language (SPL)** is an esoteric, natural programming language built entirely using the beautiful and culturally rich vocabulary of the **Sylheti dialect** (সিলেটি ভাষা). Built on top of Node.js, SPL transpiles authentic Sylheti expressions directly into executable JavaScript.

Now you can code like a true Sylheti! 🚀

---

## 🚀 Installation & Setup

To use SPL globally on your computer, ensure you have [Node.js](https://nodejs.org/) installed, then run:

```bash
npm install -g sylheti-programming-language

```

_(Note: If you changed the name during publishing, replace `sylheti-programming-language` with your published package name)._

---

## 💻 How to Run Your First Code

1. Create a file named `test.sylheti` on your computer.
2. Write some valid Sylheti code inside it:

```sylhetilang
ধর তনয় = ২০

যদি (তনয় সমান ২০) {
    চিল্লাইয়া_কও "উই দেখছ নি! মিলিগল!"
} নাঅইলে {
    কও "মিলল না রে বা!"
}

```

3. Open your terminal in that folder and execute it:

```bash
spl test.sylheti

```

## 📚 Complete Keyword Dictionary (A to Z)

Here is the complete mapping of SPL keywords to standard JavaScript/Programming concepts:

### 📦 Variables & Constants

| Sylheti Keyword | JS/Standard Equivalent | Meaning / Usage           |
| --------------- | ---------------------- | ------------------------- |
| `ধর` / `রাখ`    | `let` / `var`          | Declare a variable        |
| `পাক্কা`        | `const`                | Declare a constant        |
| `হাছা`          | `true`                 | Boolean True              |
| `মিছা`          | `false`                | Boolean False             |
| `খালি` / `নাই`  | `null` / `undefined`   | Empty or undefined values |

### 🔀 Conditionals & Loops

| Sylheti Keyword | JS/Standard Equivalent | Meaning / Usage   |
| --------------- | ---------------------- | ----------------- |
| `যদি`           | `if`                   | If condition      |

| `যতক্ষণ`        | `while`                | While loop        |
